製作期間：1.5か月程度

ゲーム概要...
ランダムな選択肢から行動を選択し、敵を倒すゲームです。
選択する行動によって、さまざまな効果が起こるので、それを活用して敵を倒します。

また、選択肢についているじゃんけんの勝敗によって、受けるダメージや与えるダメージが
増減します。
